Email Marketing
M
Mailchimp Not connected
Go to Mailchimp → Account → Extras → API keys → Create a key. Then find your List ID under Audience → Settings → Audience name and defaults.
K
Klaviyo Not connected
Go to Klaviyo → Settings → API Keys → Create a Private API Key with Full Access. Copy the key starting with pk_.
SG
SendGrid Not connected
Go to SendGrid → Settings → API Keys → Create API Key. Select Full Access or at minimum Mail Send + Marketing permissions.
CK
ConvertKit Not connected
Go to ConvertKit → Settings → Advanced → API. Copy your API Key (not the secret). It looks like a long string of letters and numbers.
E-Commerce
SH
Shopify Not connected
Go to Shopify Admin → Settings → Apps and sales channels → Develop apps → Create an app. Under Admin API access scopes enable read_products and read_orders. Install the app, then copy the Admin API access token.
Productivity & CMS
GS
Google Sheets Not connected
Go to Google Cloud Console → APIs & Services → Credentials → Create credentials → API key. Then enable the Google Sheets API in the library. You also need the Sheet ID from the URL of your spreadsheet: docs.google.com/spreadsheets/d/[SHEET-ID]/edit.
N
Notion Not connected
Go to notion.so/my-integrations → New integration. Give it a name, select your workspace, and copy the Internal Integration Secret. Then in your Notion database, click ⋯ → Connections → Add connection and select your integration. Copy the database ID from its URL.
Analytics
Google Analytics 4 Not connected
Uses a Service Account — no OAuth login needed. Go to Google Cloud Console → IAM → Service Accounts → Create. Grant it Viewer access to your GA4 property. Download the JSON key. Then go to GA4 → Admin → Property Access Management and add the service account email as a Viewer. Finally find your GA4 Property ID (the number under Admin → Property Settings).

Your key is stored locally in your browser and never sent anywhere except directly to Google's API.
